Writeup for RSABC

Category: Crypto
Author: Eryk

After opening the file, I realised that it was just some basic RSA encryption (although this way my first time actually doing an RSA challenge). I used http://www.factordb.com to factorise my n value and it worked! I got a Python script and got the flag!

Note: If you decrypted with message as it was, the last character is a fullstop, so I just changed it into a } and it worked!

Flag: tjctf{BOLm1QMWi3c}

The script:

from Crypto.Util.number import inverse
#http://www.factordb.com

e=65537
n=57772961349879658023983283615621490728299498090674385733830087914838280699121
c=36913885366666102438288732953977798352561146298725524881805840497762448828130

p=202049603951664548551555274464815496697
q=285934543893985722871321330457714807993

phi = (p -1)*(q - 1)
d = inverse(e, phi)

m = pow(c, d, n)
flag = hex(m)[2:-1]
print(bytes.fromhex(flag[:-1]).decode('utf-8')+"}")
