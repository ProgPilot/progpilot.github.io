Writeup for Ling Ling

Category: Forensics
Author: Tom

Ling Ling was a challenge that formed part of the 2020 TJCTF. We're provided with an image and a link to this Reddit post of that same image.

If we download each of these images, we can compare them using the sum command on Linux which calculates checksums of files. If they match, they're the same image.

They do not match, meaning that somewhere in the image there is extra data.

The first thing I thought to do was check EXIF data in both images. Reddit admins have stated in the past that Reddit removes EXIF data from images, so all we need to do is run exiftool on the meme provided with the challenge.
