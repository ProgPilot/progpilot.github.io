Writeup for Zipped up

Category: Miscellaneous
Author: Eryk

This challenge contained nested zipped files with some .txt files. After a bit of manual checking, I realised that there are 4 types of compressed files being used. These are: .zip, .bz2, .kz3 and .gz. I realised that all of these files can be unzipped with Python, soooooooooooo I started writing a script. It checked for the file extension, and then unzipped it to a folder, changed directories into the next folder and the cycle continued. In the end, I had 1001 folders (thank God all of the files were in a folder, this made this challenge 100x easier), so then I ran another script, which got all of the .txt files from all of the folders and compared all of them, as all of the fake .txt contained a fake flag: tjctf{n0t_th3_fl4g}, and after a minute, I got the flag. Flag: tjctf{p3sky_z1p_f1L35}
