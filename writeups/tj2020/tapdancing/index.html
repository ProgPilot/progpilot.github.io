Writeup for Tap Dancing

Category: Crypto
Author: Eryk

After initially opening the file, I thought it was some sort of new cipher, so I googled "Tap Code Cipher" and there is in fact such a thing! The only problem was that I couldn't associate the message with this cipher, I tried converting the 1's and the 2's into dots with the same length, and then something clicked. What if the 1's represent .'s and the 2's represented -'s? So I wrote a Python script which put spaces instead of the 0's and then changed the values. This gave me morse code, and it was actually the other way around. The 1's were -'s and the 2's were .'s. I slapped that morse into CyberChef and got the flag!

Flag: tjctf{M0RSEN0TB4SE3}

The script:

--- CODE.TXT ---
1101111102120222020120111110101222022221022202022211

--- DECRYPT.PY ---
with open("code.txt") as file:
    for i in file:
        code = i.replace("0", " ")

c1 = code.replace("1","-")
c2 = c1.replace("2",".")
print(c2)
