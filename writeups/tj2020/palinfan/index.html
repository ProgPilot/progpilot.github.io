Writeup for Sarah Palin Fanpage

Category: Web
Author: Tom
Tags: cookies

Sarah Palin Fanclub was a challenge that formed part of the 2020 TJCTF. We are given a website that showcases "Alaska's most famous governor".

The Top 10 moments page has various videos you can like, but you can only like 5 at a time.

The VIP area tells us that we need to like all 10 of the moments to get access.

In the browser cookies, there is a Base64 encoded JSON array with boolean values mapped to numbers. We can assume that this tracks our likes, so if we set every boolean to true, re-encode the JSON structure and set it as our cookie we get access to the VIP area.
