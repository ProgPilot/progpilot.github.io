Writeup for Ezoterik

Category: Forensics
Author: Eryk

This was quite an annoying one. The challenge name translated to Esoteric, meaning that some questionable languages are going to be here. For this challenge I used EsoLangs to find all of the languages (https://www.esolangs.org). When we first open the file we see a massive string of "-+<>[]". This is a language that's called BrainFuck. If we manually decode the message we just get "Yeah, no. Sorry.". After slamming my head on my keyboard for 10 minutes straight I opened the image in a Hex editor and at the very bottom we found some encypted message. It was encrypted in Base58 so after decryption we get what looks like a function. It was in fact another esoteric language called elevator. The script that was decoded basically returned ascii values of characters, which was the flag. Flag: rtcp{Not quite normal stego_4xbb45gytj}

You can see the image included with the challenge here, along with the Elevator source code.
