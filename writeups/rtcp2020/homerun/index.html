Writeup for HOOOOOME RUNNNN!!!!

Category: Crypto
Author: Tom

Home Run, as it will now be called, was a challenge which formed part of the 2020 Riceteacatpanda CTF. The challenge description detailed how Jake is "rounding the bases", and how he's passing bases 32 and 64, and that there's just "one more base to do ;D". We're also given some ciphertext - Ecbf1HZ_kd8jR5K?[";(7;aJp?[4>J?Slk3<+n'pF]W^,F>._lB/=r.

If you look up a list of binary to text encoding algorithms (like this one here on Wikipedia), you can see that the next encoding algorithm after 32 and 64 (that can accept text) is Base85, also known as Ascii85. Cyberchef has a decoding option for this, which you can see here
