Writeup for Snakes

Category: Binary/Excecutable
Author: Eryk

To get the flag in this challenge, you need to raise an exception. The easiest way of doing is to enter “0” in the main menu, which will give you the flag: rtcp{Sn^k3s_4lwAyz_g3T_F0uNd}, but be creative; there are a lot of ways to raise an exception in the program so explore!
